# Usage
```js
<ToolIcon tool="pickaxe" class=""/>
```
Which will render as <ToolIcon tool="pickaxe" class=""/>.

# Props
| Name  | Description                                   | Type                                              | Default   |
|-------|-----------------------------------------------|---------------------------------------------------|-----------|
| tool  | The tool icon to display.                     | String (pickaxe, axe, shovel, hoe, sword, shears) | Required  |
| class | Additional classes to apply to the component. | String                                            | Undefined |
